## What's New

### Dual-View Dive Profile Table
On mobile, a toggle button now appears in the Dive Profile header next to the TXT and PDF buttons. Tap it to switch between:

- **Table view** (default) — compact scrollable table, all 7 columns visible, great for a quick overview
- **Card view** — each row expands into a card with labeled fields, easier to read one stop at a time

Your preference is saved automatically.

### Cleaner Phase Column
The Phase column now shows symbols only — no text labels. Cleaner and more compact on all screen sizes.

### Gas Switch Icon
The gas switch row now shows `⇄` in the Phase column, consistent with all other row types.

### PDF Phase Column Fix
Phase column icons are now properly centered in the PDF export.
