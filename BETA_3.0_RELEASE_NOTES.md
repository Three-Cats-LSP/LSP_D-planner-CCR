## What's New

### CNS O₂ Tracker
Track your oxygen toxicity in real time. Enter depth, gas mix, bottom time and number of dives — the app calculates ppO₂, CNS% per dive, CNS% for the day, and OTU load. Colour-coded bars turn from green to yellow to red as you approach limits.

### GF Curve Visualizer
See your gradient factors as a curve. After running a Bühlmann deco plan, switch to the GF Curve tab to see the envelope from GF Low at the first stop through to GF High at the surface, overlaid with your actual ascent profile and deco stops.

### Professional PDF Export
One tap to export your full dive plan as a 4-page PDF:
- Page 1: Stats + full colour-coded dive profile table
- Page 2: Dive profile graph
- Page 3: Tissue saturation (all 16 compartments)
- Page 4: GF curve
Every page has a running header (depth · BT · gas · GF · algorithm · date) and footer (safety warning · date).

### Dive Profile Graph
Appears automatically after running any dive plan — a visual depth-vs-time graph with colour-coded stop zones, ppO₂ halos on dots, and a gradient fill.

### Redesigned Dive Profile Table
Columns: **Phase | Depth | Stop | Run | Mix | PPO₂ | CNS%**  
Every row shows time at depth, gas mix, ppO₂, and CNS contribution. Descent and bottom rows added. Icons only in the Phase column. Mobile-friendly card layout on small screens.

## How to Use

**CNS O₂:** Click the CNS O₂ tab → enter values → read results.  
**GF Curve:** Run a Bühlmann deco plan → click GF Curve tab.  
**PDF:** Run a Bühlmann deco plan → click ⬇ PDF in Deco Schedule.  
**Dive Graph:** Run any plan → graph appears automatically.
