📦 WHAT'S INCLUDED IN THIS RELEASE

✅ DIVE PLANNING FEATURES
  • Rec (Recreational Dive Planner) - PADI lookup tables
  • Bühlmann ZH-L16C - 16 tissue compartment model
  • Multi-Dive Planning - Up to 4 consecutive dives
  • NDL Tables & Deco Calculations
  • Gradient Factors (Baker safety model)

✅ USER INTERFACE
  • Professional LSP D-PLANNER branding
  • Responsive design (all screen sizes)
  • Dark/Light theme support
  • Tab navigation (Dive Planner, Multi-Dive, Reference)
  • Algorithm switcher (Rec ↔ Bühlmann)

✅ INSTAGRAM INTEGRATION
  • Square gradient icon (bottom center)
  • Scannable QR code in Reference section
  • Direct link to @threecats_lsp profile
  • Professional Instagram gradient colors

✅ DEPLOYMENT OPTIONS
  • Web version (HTML5, PWA compatible)
  • Android APK (full native app)
  • GitHub Actions (automated builds)

📂 HOW TO USE THIS BASELINE

This is your STABLE FOUNDATION for future versions. Here's how to proceed:

FOR FUTURE DEVELOPMENT:
  1. Always use this release as your starting point
  2. Update version numbers when making changes
  3. Keep three files in perfect sync:
     • index.html (main source)
     • www/index.html (web copy)
     • android/app/src/main/assets/app.html (APK copy)
  4. Create new RELEASE_[VERSION].md for each update
  5. Test in browser, PWA, and APK before release

BRANCH STRATEGY:
  • main = Production ready (current: Beta 1.1)
  • develop = Active development
  • feature/* = Individual features
  • release/* = Release candidates
