# LSP D-PLANNER

A dive planning tool for recreational and technical divers, built by Three Cats LSP Diving Team.

> ⚠️ **IMPORTANT — READ BEFORE USE**
> This application is a planning aid only. It does not replace proper dive training, certification, or a calibrated dive computer. See [Disclaimer](#disclaimer) below.

## What it does

- **Recreational planning** — PADI RDP-based NDL lookup, multi-dive day planning with nitrogen tracking
- **Bühlmann ZH-L16C + GF** — full decompression schedule generation with gradient factors, multi-gas support, tissue saturation display
- **Tools** — MOD, Best Mix, Max Depth, Average Depth converter, Unit Converter
- **Solver** — interactive surface interval / bottom time trade-off explorer

## Algorithm

Uses Bühlmann ZH-L16C with Erik Baker's Gradient Factor modification. Key implementation details:

- Baker GF ceiling formula: `pAmbMin = (pN2 − gf·a) / (1 − gf + gf/b)`
- ZHL-16C canonical constants (ht=4.0 min for compartment 1, verified against ApexDeco and DiveProMe source)
- Alveolar water vapour subtracted from inspired gas (0.0627 bar at 37°C)
- Schreiner equation for linear pressure changes during descent/ascent
- 10-second stop resolution

Plans have been compared against Multideco 4.19, ApexDeco, DiveProMe, Decosoft, and Divesoft. Typical agreement is within 1–2 minutes per stop. Differences between reference implementations are expected — no two planners produce identical output for the same inputs.

## How to use

Download `index.html` and open it in any modern browser. No installation, no server, no internet required after download.

Works offline. Can be added to your phone home screen as a PWA.

## Disclaimer

**This tool is for planning and educational purposes only.**

- It is **not** a certified dive computer
- It is **not** a substitute for professional dive training or certification

- Decompression algorithms are mathematical models — they do not account for individual physiology, fitness, age, hydration, thermal state, or workload
- Never dive to the exact limits of any table or planner

Dive planning is a skill taught in dive courses. If you do not hold a relevant certification for the type of diving you are planning, **get trained first**.

## GF presets

| Preset | Setting | Notes |
|--------|---------|-------|
| Conservative | GF 30/70 | Deep first stop, maximum deco obligation |
| Moderate | GF 40/80 | Balanced |
| Recommended | GF 70/85 | Dr. David Doolette's recommendation following NEDU TR 2011-06 research on deep stops |

The NEDU 2011 study found that redistributing deco time from shallow to deep stops increased DCS incidence (11/198 vs 3/192, p=0.030). GF 70/85 avoids excessive deep stops while maintaining a conservative surface limit.
