# Quick Start — 5 Minute Web Deployment

## Deploy Web App to Netlify (Instant!)

### Fastest Way (No Login)

1. Go to: **https://app.netlify.com/drop**
2. Drag **`www/index.html`** file here
3. Done! You get a live URL instantly
4. Share the URL with anyone

**That's literally it. Your app is live in 10 seconds.**

---

## Want APK for Android? (20 minutes)

```bash
# Push to GitHub (or just drag files to GitHub)
git init
git add .
git commit -m "Initial"
git remote add origin https://github.com/YOUR-USERNAME/dive-planner.git
git push -u origin main

# Go to Actions tab → Build APK → Run workflow
# Wait 20 minutes
# Download APK from artifacts
```

---

## Both Versions Working!

✅ **Web app**: https://[your-netlify-url].netlify.app
✅ **Android APK**: Download from GitHub Actions
